### CSS 变量

| CSS 变量 | 描述 |
| --- | --- |
| \-\-rate-color | 图标未选中颜色 |
| \-\-rate-color-selected | 图标选中颜色 |
| \-\-rate-size | 图标大小 |
| \-\-rate-gap | 图标间距 |
| \-\-rate-popover-color | 提示文字颜色 |
| \-\-rate-popover-text-size | 提示文字大小 |
| \-\-rate-popover-text-height | 提示文字行高 |
| \-\-rate-popover-radius | 提示框圆角 |
| \-\-rate-popover-padding | 提示框内边距 |
